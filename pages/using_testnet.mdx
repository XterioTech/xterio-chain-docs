## Add Xterio Testnet to MetaMask

### Using the Xterio Chain Rollup Info Page

1. Navigate to our [Rollup Info Page](https://rollup-info.altlayer.io/best-garment-primary/xterio_testnet)
2. Click on the "**Add to Wallet**" button

<img src="/addwallet_testnet.jpg" width="500" />

### Manually add via MetaMask

You can add the network manually using the above-mentioned network details.
