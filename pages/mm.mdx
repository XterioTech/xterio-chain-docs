# Add Xterio Chain to Metamask

## Using the Xterio Chain Rollup Info Page

1. Navigate to our Rollup Info Page: [Mainnet](https://rollup-info.altlayer.io/garment-primary-minute/xterio) / [Testnet](https://rollup-info.altlayer.io/best-garment-primary/xterio_testnet)
2. Click on the "**Add to Wallet**" button

![image](./images/addwallet.jpg)

## Manually add via Metamask

You can add the network manually using the following details:

### Mainnet

| Parameter | Value |
| --- | --- |
| Network Name | Xterio Chain |
| RPC URL | https://xterio.alt.technology |
| Chain ID | 112358 |
| Currency Symbol | BNB |
| Block explorer URL | https://xterscan.io |
